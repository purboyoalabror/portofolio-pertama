Introduction

HTML stands for HyperText Markup Language.

It is the standard language used to create web pages.

HTML describes the structure of a web page using markup.

HTML consists of a series of elements that tell the browser how to display content.

Web browsers do not display the HTML tags but use them to render the content.

<!DOCTYPE html> <html></html> <head></head> <body></body>
  • HTML is not a programming language.
  • It is a markup language.
  • Tags are the building blocks of HTML.
HTML Elements

An HTML element is defined by a start tag, some content, and an end tag.

Elements can contain text, other elements, or both.

  • Heading tags: <h1> to <h6>
  • Paragraph: <p>
  • Break: <br>
HTML Attributes

Attributes provide additional information about elements.

They are always specified in the start tag.

Common attributes include `href`, `src`, `alt`, `id`, and `class`.

<a href="https://example.com">Visit</a> <img src="image.jpg" alt="Image">
  • `href` defines the URL of a link.
  • `src` specifies the path to an image.
  • `alt` provides alternate text for images.
HTML Forms

Forms are used to collect user input.

A form element contains input elements like text fields, checkboxes, and buttons.

Each form control is defined with a unique tag.

<form></form> <input type="text">
  • <input> for text fields
  • <label> for field labels
  • <button> to submit forms
HTML Semantics

Semantic elements clearly describe their meaning to both the browser and developer.

Examples include <article>, <section>, <nav>, <header>, <footer>.

They help with SEO and accessibility.

<section>This is a section</section>
  • <article> for self-contained content
  • <nav> for navigation links
  • <footer> for page footer
